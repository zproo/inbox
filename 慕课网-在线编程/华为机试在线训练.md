# 华为机试在线训练

## 第一页

### 1.字符串最后一个单词的长度

```java
import java.util.Scanner;

public class Main {

    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        String s = "";
        while (input.hasNextLine()) {
            s = input.nextLine();
            String[] split = s.split(" ");
            System.out.println(split[split.length - 1].length());
        }
    }
}
```
